## 📚 项目介绍

这是一个包含 100 道算法题的集合，每道题都有详细的中英文解释、使用场景、算法难度、时间复杂度、空间复杂度和测试代码。这些算法题涵盖了各种常见的算法类型，适合算法学习和面试准备。

### 🎯 项目特色

- **详细的算法解释**: 每道题都有详细的中英文算法解释
- **完整的测试代码**: 每个文件都包含 `if __name__ == "__main__"` 块和测试代码
- **多种算法类型**: 涵盖数组、链表、字符串、二叉树、动态规划等多种算法类型
- **不同难度等级**: 包含简单、中等、困难三个难度等级的题目
- **清晰的文件结构**: 每个算法题都有独立的文件，命名合理
- **易于使用**: 可以直接运行每个文件来测试算法的正确性

## 🛠️ 技术栈

- **语言**: Python 3
- **开发环境**: 任何支持 Python 3 的环境
- **测试方法**: 每个文件都包含 `if __name__ == "__main__"` 块和测试代码
- **版本控制**: Git

## 📁 算法类别分类

### 1. 数组操作
- **简单**: 两数之和、移除元素、删除排序数组中的重复项
- **中等**: 三数之和、四数之和、盛最多水的容器、接雨水
- **困难**: 寻找两个正序数组的中位数

### 2. 链表操作
- **简单**: 合并两个有序链表、反转链表
- **中等**: 两数相加、删除链表的倒数第 N 个节点、链表相交、环形链表
- **困难**: 合并 K 个升序链表、K 个一组翻转链表

### 3. 字符串操作
- **简单**: 最长公共前缀、有效的括号、回文数、罗马数字转整数
- **中等**: 最长回文子串、最长无重复字符的子串、电话号码的字母组合
- **困难**: 编辑距离

### 4. 二叉树操作
- **简单**: 二叉树的最大深度、二叉树的最小深度、对称二叉树、相同的树
- **中等**: 二叉树的层序遍历、二叉树的中序遍历、二叉树的前序遍历、二叉树的后序遍历、二叉树的右视图、二叉树的左视图、路径总和、路径总和 II、路径总和 III
- **困难**: 二叉树中的最大路径和、从前序与中序遍历序列构造二叉树、从中序与后序遍历序列构造二叉树

### 5. 动态规划
- **简单**: 爬楼梯、最大子序和、买卖股票的最佳时机
- **中等**: 最小路径和、单词拆分
- **困难**: 买卖股票的最佳时机 III、买卖股票的最佳时机 IV

### 6. 回溯算法
- **中等**: 组合总和、组合总和 II、全排列、全排列 II、子集、子集 II、单词搜索
- **困难**: 电话号码的字母组合

### 7. 双指针技巧
- **中等**: 三数之和、四数之和、盛最多水的容器、接雨水
- **困难**: 寻找两个正序数组的中位数

### 8. 二分查找
- **简单**: 二分查找、搜索插入位置、x 的平方根
- **困难**: 寻找两个正序数组的中位数

### 9. 哈希表
- **简单**: 两数之和、有效的字母异位词
- **中等**: 最长无重复字符的子串

### 10. 贪心算法
- **简单**: 买卖股票的最佳时机 II
- **中等**: 跳跃游戏、跳跃游戏 II
- **困难**: 买卖股票的最佳时机 III、买卖股票的最佳时机 IV

## 🚀 如何使用

### 1. 克隆仓库

```bash
git clone https://github.com/Geekmister/AlgorithmProblem.git
cd AlgorithmProblem
```

### 2. 运行算法题

每个算法题都可以独立运行，例如：

```bash
python python/two_sum.py
```

### 3. 测试算法

每个文件都包含测试用例，可以直接运行文件来验证算法的正确性。

### 4. 查看算法详情

每个算法题文件都包含详细的中英文解释、使用场景、算法难度、时间复杂度、空间复杂度和测试代码。例如，查看两数之和的算法详情：

```bash
cat python/two_sum.py
```
